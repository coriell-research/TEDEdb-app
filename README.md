## TEDEdb-app

This repository contains the code for the "Transposable Element Differential Expression Database" 
Shiny application. The app is organized into modules for each page. A 'data' directory containing 
the resources needed to run the app is located in the [appdata](https://data.coriell.org/TEDEdb/appdata/) directory of the database FTP server. The DESCRIPTION file lists required dependencies.
